Pasir

En este proyecto se ampliarán los conocimientos enseñados durante el Grado Superior de Administración de Sistemas Informáticos en Red. También se aprovecharán lo aprendido para implementarlo y profundizar todo lo posible. El proyecto se basa en un entorno de trabajo para una empresa PYMES, donde crearemos un cluster a partir de 3 Servidores PROXMOX, y donde se crearán las máquinas virtuales y contenedores necesarios donde se instalarán y configurarán los servicios que se usarán para el proyecto.
Dichos servicios serán:

  • Servidor GitLAB.
  • Contenedor con Docker, Docker-Compose y nodo Master de Ansible para automatizar tareas y despliegues. En este contenedor también instalaremos GitLAB Runner para desplegar contenedores Docker desde GitLAB.
  • Servidor de Correos.
  • Servidor Web con JavaScript para el uso de esta misma página web.
  • Servidor Web con base de datos y PHP para almacenamiento de datos y muestra de ellos a través de Frontend.
